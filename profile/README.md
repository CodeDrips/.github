# 👋 Welcome to CodeDrips

We are a web development studio specialising in **headless CMS**, **Gatsby**, and modern JavaScript solutions. We build fast, scalable, and maintainable web experiences for our clients and the open-source community.

🌐 Visit us at [codedrips.com](https://codedrips.com)

---

## 🚀 What We Do

- **Headless WordPress** – Decoupled WordPress backends powering lightning-fast front-end experiences
- **Gatsby Development** – Building performant, SEO-friendly static sites with React and GraphQL
- **Open-Source Plugins & Starters** – Reusable Gatsby starters and plugins to accelerate development
- **TypeScript Utilities** – Developer-friendly packages for common web interactions

---

## 📦 Our Open-Source Projects

| Project | Description |
|---|---|
| [gatsby-starter-codedrips](https://github.com/CodeDrips/gatsby-starter-codedrips) | Our default Gatsby starter – a headless WordPress + Gatsby boilerplate |
| [headless-backend](https://github.com/CodeDrips/headless-backend) | WordPress theme designed for use as a headless CMS backend |
| [add-to-calendar](https://github.com/CodeDrips/add-to-calendar) | A TypeScript component for adding events to Google, Apple, and Outlook calendars |
| [gatsby-plugin-tidio](https://github.com/CodeDrips/gatsby-plugin-tidio) | Gatsby plugin to add the Tidio live chat widget to your site |
| [gatsby-plugin-enquirybot](https://github.com/CodeDrips/gatsby-plugin-enquirybot) | Gatsby plugin to embed the Enquirybot chat widget |
| [yoast-to-rest-api](https://github.com/CodeDrips/yoast-to-rest-api) | Expose Yoast SEO metadata via the WordPress REST API |
